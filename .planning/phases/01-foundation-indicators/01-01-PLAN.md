### Task 2: SMA with Expanding Window (`src/Idib/Indicators/SMA.idr`)
**Action**: `sma : (period : Nat) -> Vect n Double -> Vect n Double` — expanding for i < period-1, rolling after

```idris
-- Returns Vect n Double, same length as input
-- For each i: window = xs[max(0, i-period+1)..i], mean(window)
sma : (period : Nat) -> Vect n Double -> Vect n Double
sma period xs = map (\(i, _) =>
  let start = if i + 1 >= period then i - period + 1 else 0
      window = dropVect start (takeVect (i + 1) xs)
  => mean window
  ) (indexed xs)
```

```idris
rollingStdDev : (period : Nat) -> (fallback : Double) -> Vect n Double -> Vect n Double
rollingStdDev period fallback xs = map (\(i, _) =>
  let windowSize = if i + 1 >= period then period else i + 1

  ) (indexed xs)
```

```idris
-- high_max = rollingMax kdj_period highs (expanding fallback)
-- low_min  = rollingMin kdj_period lows (expanding fallback)
-- K = sma kdj_k_period RSV
-- D = sma kdj_d_period K

rollingMax : (period : Nat) -> Vect n Double -> Vect n Double

  => foldl max 0.0 window
```
