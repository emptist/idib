# Phase 04-01: Multi-Timeframe Architecture

## Objective
Enable multi-timeframe strategy (e.g., monthly trend filter + daily entry) and multi-chart display. Types enforce alignment across intervals.

## Context
- Reference: glib's single-interval `evaluate_bar` + user requirement for multi-TF
- Depends on: Phase 01-03 (Indicators, Strategy, Signal Series)
