## Key Insight: SimpleFish vs Fish

### SimpleFish = Basic Alternating Segments

SimpleFish is the **real model** for the existing xxx7 series:

| SimpleFish | Indicator | What It Computes |
|------------|-----------|------------------|
| YangSimpleFish boundary | `hprd7` | New peak in SMA7 series |
| YinSimpleFish boundary | `lprd7` | New trough in SMA7 series |
| Bars within YangSimpleFish | `hrows7` | Bar index within peak group |
| Bars within YinSimpleFish | `lrows7` | Bar index within trough group |
| Mean within YangSimpleFish | `cmah7` | Expanding mean within peak group |
| Mean within YinSimpleFish | `cmal7` | Expanding mean within trough group |
| Mean within nested SimpleFish | `hlcmah7` | Expanding mean within nested segments |
| SimpleFish stability | `cnst7` | Expanding mean of (rising && hlhrows == 0) |

Each "run" in hprd/lprd is a SimpleFish — an alternating sequence of peaks and troughs.

### Fish = Cross-SimpleFish Segments

Fish is a **larger segment** that crosses multiple SimpleFishes:
- A Fish gives the **whole rising or dropping segment** in each "run"
- A Fish could be composed of multiple SimpleFishes
- Example: A YangFish might contain several YangSimpleFish + YinSimpleFish pairs

```
Fish (whole rising segment)
  ├── YangSimpleFish 1 (hprd run)
  │     └── YinSimpleFish (nested)
  ├── YangSimpleFish 2 (hprd run)
  │     └── YinSimpleFish (nested)
  └── YangSimpleFish 3 (hprd run)
```

### Recursive Structure

SimpleFish are **fractal** — inside a SimpleYinFish, there can be smaller SimpleYangFish:
- `hprd7` → detect primary SimpleFish (YangSimpleFish boundaries)
- `hlprd` → detect nested SimpleFish within each primary SimpleFish
- `hlhprd` → detect doubly-nested SimpleFish within each nested SimpleFish

This is exactly what the indicator chain computes in Vector.idr.

### New Fish with N-Bar Confirmation

For production use (not just SMA7), Fish can have N-bar confirmation:
- N = 2 × barsPerMonth(interval): 1mo=2, 1wk=8, 1d=40, 1h=260, 4h=64
- **Disabled for sub-hourly** (Min1/5/15/30 return 0 barsPerMonth)
- YangFish: TRUE start = lowest in subsequent YinSimpleFish (back-count at new HIGH)
- YinFish: TRUE start = highest in subsequent YangSimpleFish (back-count at new LOW)

### Key Properties
- **Parameter-free except N** (interval-derived, not tuned) — "natural" segments
- Research record for AI pattern discovery
- Alternation invariant enforced by GADT
- Recursive structure: SimpleFish within SimpleFish (fractal pattern)
