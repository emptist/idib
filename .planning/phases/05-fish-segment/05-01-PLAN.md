## Key Insight: Leaf vs Branch

### Leaf = Basic Alternating Segments

Leaf is the **real model** for the existing xxx7 series:

| Leaf | Indicator | What It Computes |
|------|-----------|------------------|
| YangLeaf boundary | `hprd7` | New peak in SMA7 series |
| YinLeaf boundary | `lprd7` | New trough in SMA7 series |
| Bars within YangLeaf | `hrows7` | Bar index within peak group |
| Bars within YinLeaf | `lrows7` | Bar index within trough group |
| Mean within YangLeaf | `cmah7` | Expanding mean within peak group |
| Mean within YinLeaf | `cmal7` | Expanding mean within trough group |
| Mean within nested Leaf | `hlcmah7` | Expanding mean within nested segments |
| Leaf stability | `cnst7` | Expanding mean of (rising && hlhrows == 0) |

Each "run" in hprd/lprd is a Leaf — an alternating sequence of peaks and troughs.

### Branch = Cross-Leaf Segments

Branch is a **larger segment** that crosses multiple Leaves:
- A Branch gives the **whole rising or dropping segment** in each "run"
- A Branch could be composed of multiple Leaves
- Example: A YangBranch might contain several YangLeaf + YinLeaf pairs

```
Branch (whole rising segment)
  ├── YangLeaf 1 (hprd run)
  │     └── YinLeaf (nested)
  ├── YangLeaf 2 (hprd run)
  │     └── YinLeaf (nested)
  └── YangLeaf 3 (hprd run)
```

### Recursive Structure

Leaves are **fractal** — inside a YinLeaf, there can be smaller YangLeaves:
- `hprd7` → detect primary Leaves (YangLeaf boundaries)
- `hlprd` → detect nested Leaves within each primary Leaf
- `hlhprd` → detect doubly-nested Leaves within each nested Leaf

This is exactly what the indicator chain computes in Vector.idr.

### Branch with N-Bar Confirmation

For production use (not just SMA7), Branches can have N-bar confirmation:
- N = 2 × barsPerMonth(interval): 1mo=2, 1wk=8, 1d=40, 1h=260, 4h=64
- **Disabled for sub-hourly** (Min1/5/15/30 return 0 barsPerMonth)
- YangBranch: TRUE start = lowest in subsequent YinLeaf (back-count at new HIGH)
- YinBranch: TRUE start = highest in subsequent YangLeaf (back-count at new LOW)

### Key Properties
- **Parameter-free except N** (interval-derived, not tuned) — "natural" segments
- Research record for AI pattern discovery
- Alternation invariant enforced by GADT
- Recursive structure: Leaf within Leaf (fractal pattern)
